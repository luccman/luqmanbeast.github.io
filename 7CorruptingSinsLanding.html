7 Corrupting Sins — Game Landing Page will be a visually immersive, responsive landing page for the game 7 Corrupting Sins, built with React, Vite, and Material UI (MUI).

Features

  • Custom MUI Theme: Unique color palette, custom fonts, button variants, and AppBar styling
  • Responsive Layout: Mobile-friendly with adaptive typography and layout
  • 3D Globe Model: Interactive 3D model rendered with react-three-fiber
  • Sectioned Content: Hero, About, Media Gallery, Factions, Pre-Order, The 7 Sins and Footer
  • Animation: Used Framer Motion for smooth transitions and animations
  • Accessible & Semantic: Semantic HTML and accessible MUI components

Tech Stack

  • React 19
  • Vite
  • Material UI (MUI)
  • Framer Motion
  • @mui/lab Timeline
  • react-three-fiber
  • @emotion/styled
  • Custom fonts via @font-face

Customization

  • Fonts: Add/replace font files in public/assets/fonts/ and update src/fonts.css
  • Theme: Edit src/theme.jsx for colors, typography, or component variants
  • 3D Model: Replace public/assets/stylized_planet.glb with your own model
  • Images: Add images to public/assets/ and update component imports
  • Animations: Modify Framer Motion animations in components

Notable Components

  • Navbar: Custom AppBar with textured background and themed navigation buttons
  • Hero: Large banner with background image, gradient overlay, and call-to-action buttons
  • About: Split layout with video and lore/description
  • Factions: 3D globe and dual timelines for Humans and Jinns
  • The 7 Sins: Individual sin component with rotating selector
  • Footer: Social links and copyright
