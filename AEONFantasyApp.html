Status: IN PROGRESS

AEON Fantasy App will be a modern, cross-platform mobile application for AEON Fantasy arcade users, built with React Native, Expo, and TypeScript. It enhances the arcade experience by providing features like user profile management, arcade discovery, activity tracking, and more—all in a playful, user-friendly interface.

Features

  • User Profile: View/edit profile, interactive tier badge demo, quick email contact
  • Arcade Discovery: Browse/filter arcades, see nearest arcades with distance/address, arcade cards with logos
  • Activity Tracking: Visualize spending with a donut chart (Victory Native), list of recent activities
  • Custom Navigation: Bottom tab navigation with custom icons and playful design
  • Settings & Preferences: Change password, manage payment, language, privacy, and log out
  • Responsive Design: Works on iOS, Android, and web (via Expo)

Tech Stack

  • React Native (Expo)
  • TypeScript
  • Victory Native (charts)
  • React Navigation & Expo Router
  • Custom SVG/PNG assets
  • Styled with StyleSheet

Notable Implementation Details

  • Profile Tier Demo: Tier badge next to avatar is interactive—tap to cycle tiers and points
  • Charts: Activity spending visualized with Victory Native Pie/PolarChart
  • Custom Tab Bar: Bottom navigation uses custom images and playful layout
  • TypeScript: Fully typed for reliability and maintainability

Customization & Extensibility

  • Assets: Swap icons/images in assets/ to match your brand
  • Navigation: Easily add new screens/tabs via Expo Router
  • Theming: Centralized in constants/Colors.ts for easy updates
