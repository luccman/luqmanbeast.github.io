Project Overview

This project showcases the UI/UX design for an All-in-One AEON Fantasy Customer App. The app is designed to provide a seamless and engaging experience for AEON Fantasy customers, integrating features such as membership management, promotions, store locator, and interactive games.

Tools Used: Figma (for wireframes, high-fidelity designs, and interactive animations)

Wireframes & Design Process

The design process began with low-fidelity wireframes to map out the user journey and core features. These were iterated into high-fidelity mockups with a playful and modern aesthetic, reflecting the AEON Fantasy brand.

Interactive Animation

Using Figma's prototyping and animation tools, I created interactive flows to demonstrate the app's fluid navigation, micro-interactions, and engaging transitions.

Key Features

  • Modern, playful interface tailored for families and children
